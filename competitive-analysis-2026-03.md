## 3. 逐一競爭者深度分析

### 3.1 AutoReel — 威脅：高

| 項目 | 分析 |
|------|------|
| 核心功能 | 照片→AI 電影級運鏡影片，支援從 Zillow/Realtor.com 直接匯入，drone-style flythrough |
| 定價 | Essential $396/年，Pro $790/年，前 3 支免費 |
| 強項 | 最成熟的照片轉運鏡影片技術；MLS 整合；攝影師可轉售給客戶（$50-100/支） |

## 9. 策略建議

### ReelEstate 的核心競爭優勢

1. **無人做到的完整度**：照片→AI 運鏡（Kling V2.5）→虛擬裝潢（nano-banana-2）→TTS 旁白→自動剪輯→字幕→成品影片。競爭者最多覆蓋其中 2-3 個環節。
2. **台灣市場真空**：搜尋結果中零家台灣本土 AI 房產影片工具。這是先行者優勢窗口。
3. **技術差異化**：使用 Kling V2.5 首尾幀生成真實運鏡影片，而非競爭者常用的 2.5D depth parallax（Ken Burns 效果升級版）。前者產出更接近真實攝影。
4. **通路策略**：Telegram Bot → 零學習曲線，比 Web dashboard 更適合台灣房仲的工作習慣。

### 建議優先行動

- **短期**：快速取得 5-10 個台灣房仲測試用戶，建立 before/after 案例集
- **中期**：考慮 LINE Bot 入口（台灣 LINE 滲透率 > Telegram）
- **長期**：B2B OEM 白標給信義/永慶等加盟體系，以通路鎖定市場
